# Zivo Creative Website

A premium video editing agency website built with React, Vite, and Tailwind CSS.

## Features

-   **Modern UI:** Sleek, dark-themed design with glassmorphism effects.
-   **Tubelight Navbar:** Interactive floating navigation bar with glowing effects using Framer Motion.
-   **Responsive Design:** Fully optimized for desktop and mobile devices.
-   **WhatsApp Integration:**
    -   Direct WhatsApp links for "Get Started" and "Book a Call".
    -   Dynamic pre-filled messages for Services and Pricing packages.
    -   Contact form submission redirects to WhatsApp with filled details.
-   **Pricing Section:** Clear package listings (Basic, Standard, Premium) with detailed feature breakdowns.

## Tech Stack

-   **Frontend:** React, TypeScript, Vite
-   **Styling:** Tailwind CSS, Lucide React (Icons)
-   **Animations:** Framer Motion
-   **Routing:** React Router

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/DEViLxPLAYS/Kivomediawebsite.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run development server:
    ```bash
    npm run dev
    ```

## Build

To create a production build:
```bash
npm run build
```
